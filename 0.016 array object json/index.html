Q1: Array Basics

Task: Create an array of fruits, display first and last, add and remove items.

Output:


    
Code:

fruits.shift();
console.log(fruits);

Q3: Reference vs Copy

    
Task: Demonstrate array reference and spread copy.

    
Output:

    

    
Code:

    
let arr = [1,2,3];
let arr2 = arr;
arr2.push(4);
let arr3 = [...arr];
arr3.push(5);
console.log(arr);
console.log(arr2);
console.log(arr3);

Q8: Combined Practice

    
Task: Array of student objects, map names, filter by age, convert to JSON.

    
Output:

    

    
Code:

    
let studentsArray = [
{name:'A', age:12, grade:7},
{name:'B', age:13, grade:8},
{name:'C', age:11, grade:6}
];
let namesArray = studentsArray.map(s => s.name);
let ageFiltered = studentsArray.filter(s => s.age>12);
let jsonString = JSON.stringify(studentsArray);
console.log(namesArray);
console.log(ageFiltered);
console.log(jsonString);
